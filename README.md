# Django-Skeleton

## [ Description ]

- drf로 서비스를 개발할 때 필요한 기본 설정들에 대한 레포지토리입니다.

## [ Structure ]

Root DIR

ㄴ backend : django 기반 Web Application Server

ㄴ frontend : 미정

ㄴ nginx : Nginx 기반 Web Server (Proxy)

ㄴ docker-compose : 개발, 테스트, 배포 단계에서 사용될 docker-compose 파일들

ㄴ env : docker-compose에서 활용될 환경변수
